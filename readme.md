# Controle de Frotas

Esse repositório é um controle de frotas que conta com módulos de gerenciamento de notas, rotas e funcionários.

É necessário ter instalado:

 * Mysql 5.5 
 * PHP 7.0
 * Composer
 * Laravel 5.4
